XMPP-FTW


XMPP For The Web

What?

The goal of XMPP-FTW is to make XMPP really simple to use for (web) developers

  • JSON-based API to XMPP
  • No hand-holding
  • Events driven
  • Common sense
  • Good documentation

Technology

Written in nodejs

Doesn't mandate a transport

  • Demo system uses a websocket
  • Powered by Primus


Acts as an XMPP client (using `node-xmpp`)
